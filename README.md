# 🚀 Mugixor - نظام الإدارة المتقدم

## 📋 نظرة عامة

موقع Mugixor للمركبات الكهربائية مع **نظام إدارة متكامل** يوفر إدارة شاملة للمنتجات والصور والترجمات.

## ✨ الميزات الرئيسية

### 🌐 **نظام الترجمة:**
- **6 لغات كاملة:** الإسبانية، الباسكية، الإنجليزية، الفرنسية، الهولندية، الألمانية
- **تبديل فوري** بدون إعادة تحميل
- **حفظ التفضيلات** في المتصفح
- **علم باسكي حقيقي** 🏴

### 🛠️ **نظام الإدارة:**
- **لوحة تحكم شاملة** مع إحصائيات
- **إدارة المنتجات** (إضافة/تعديل/حذف)
- **إدارة الصور** مع رفع متعدد
- **إدارة الترجمات** لجميع اللغات
- **نظام أمان** بكلمة مرور

### 📱 **التصميم المتجاوب:**
- **متوافق مع جميع الأجهزة**
- **تصميم عصري** وأنيق
- **سرعة تحميل عالية**
- **تجربة مستخدم متميزة**

## 🚀 التشغيل السريع

### المتطلبات:
- Node.js (18+)
- npm أو pnpm

### التثبيت:
```bash
# 1. استخراج الملفات
unzip MUGIXOR-ADMIN-SYSTEM-FINAL.zip
cd mugixor-admin-final

# 2. تثبيت التبعيات
npm install

# 3. تشغيل الموقع
npm run dev

# 4. فتح المتصفح
# http://localhost:5173
```

## 📊 أقسام الإدارة

### 1. **نظرة عامة (Overview)**
- إحصائيات الموقع
- عدد المنتجات واللغات
- معلومات النظام

### 2. **إدارة المنتجات (Products)**
- إضافة منتجات جديدة
- تعديل المنتجات الموجودة
- حذف المنتجات
- البحث والفلترة
- ترجمة لـ 6 لغات

### 3. **إدارة الصور (Images)**
- رفع صور متعددة
- تنظيم الصور حسب النوع
- معاينة وتحميل الصور
- حذف جماعي للصور
- نسخ روابط الصور

### 4. **إدارة الترجمات (Translations)**
- تحرير الترجمات مباشرة
- البحث في النصوص
- كشف الترجمات المفقودة
- حفظ تلقائي للتغييرات
- تصدير/استيراد الترجمات

## 🛍️ إدارة المنتجات

### إضافة منتج جديد:
1. اذهب لقسم "Products"
2. انقر "Add New Product"
3. املأ البيانات:
   - **الاسم** (6 لغات)
   - **الوصف** (6 لغات)
   - **السعر** (باليورو)
   - **الفئة** (دراجة/سكوتر/كارجو/سكيتبورد)
   - **منتج مميز** (نعم/لا)
   - **الصور** (متعددة)
   - **الميزات** (قائمة بـ 6 لغات)
4. انقر "Save"

### تعديل منتج:
1. انقر أيقونة التعديل ✏️ بجانب المنتج
2. عدل البيانات المطلوبة
3. انقر "Save"

### حذف منتج:
1. انقر أيقونة الحذف 🗑️ بجانب المنتج
2. أكد الحذف

## 🖼️ إدارة الصور

### رفع صور جديدة:
1. اذهب لقسم "Images"
2. انقر "Upload Images"
3. اختر الصور من جهازك
4. حدد نوع الصورة
5. انقر "Upload"

### إدارة الصور:
- **البحث:** استخدم مربع البحث
- **الفلترة:** اختر نوع الصورة
- **العرض:** شبكي أو قائمة
- **النسخ:** انقر أيقونة النسخ لنسخ الرابط
- **التحميل:** انقر أيقونة التحميل
- **الحذف:** اختر الصور واحذفها

## 🌐 إدارة الترجمات

### تحرير الترجمات:
1. اذهب لقسم "Translations"
2. ابحث عن النص المطلوب
3. انقر على النص لتحريره
4. عدل الترجمة
5. انقر "Save" أو اضغط Enter

### البحث والفلترة:
- **البحث:** ابحث في النصوص
- **فلترة اللغة:** اختر لغة محددة
- **النصوص المفقودة:** اعرض الترجمات الناقصة
